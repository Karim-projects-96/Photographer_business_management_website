# Project Development Roadmap

## Phase 1: Foundation (Week 1-2)
- Database schema design and MySQL table creation.
- Core Auth system (Login/Signup) and Database connection setup.

## Phase 2: Marketplace (Week 3-4)
- Homepage search engine development (Location/Category filters).
- Dynamic portfolio page generation based on user slugs.

## Phase 3: Business Tools (Week 5-6)
- PDF generation for Quotations and Contracts.
- Financial Dashboard integration with Chart.js.

## Phase 4: Automation (Week 7-8)
- Integration of Cloudinary and Google Drive APIs.
- Writing and testing Cron jobs for the 48-hour "Flash" purge.
