# Functional Product Requirements

## 1. Marketplace Search Engine
- Users must be able to search by City (String match) and Category (Dropdown).
- Search results must exclude photographers with 'Booked' status in the `availability` table for the selected date.

## 2. Dynamic Portfolio System
- Each photographer has a unique landing page: `snapbroker.com/studio-name`.
- Features: Bio, Service Packages, Image Gallery (Cloudinary), and Inquiry Form.

## 3. Financial Management (SaaS Side)
- Automated calculation of "Remaining Balance" (Total Deal - Advance Paid).
- Visual bar charts showing monthly revenue vs. business expenses using Chart.js.

## 4. Document Automation
- One-click PDF generation for Quotations based on selected packages.
- Digital E-Signature capture for legally binding contracts.
