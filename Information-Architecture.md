# Information Architecture (IA)

## User Navigation Path
1. **Home:** Search Bar -> Featured Photographers -> About Us.
2. **Search Results:** Photographer Cards -> View Portfolio.
3. **Booking Flow:** Inquiry Form -> Receive Quote -> Sign Contract -> Pay Advance.
4. **Client Portal:** View Proofs -> Select Favorites -> Pay Balance -> Download.

## Data Security Layers
- **Session Protection:** Users can only access projects linked to their ID.
- **Logic Gates:** Google Drive links remain hidden (`is_unlocked = 0`) until balance is cleared.
