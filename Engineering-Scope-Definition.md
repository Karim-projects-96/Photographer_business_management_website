# Engineering Scope

## In-Scope
- Photographer registration and verification.
- Searchable marketplace for clients.
- Automated PDF generation (Quotes/Contracts).
- Income/Expense tracking dashboard.
- Automated storage cleanup.

## Out-of-Scope
- Direct Credit Card payment processing (Manual UPI verification for MVP).
- Advanced AI image enhancement tools.
- Native mobile applications (Web-only for Phase 1).
